# HER2 expression in prostatic carcinomas – a systematic review and meta-analysis of a potential therapeutic target

## Overview

The analysis pipeline performs three meta-analytic tasks:

1. **Cross-tabulation meta-analysis** — Compares HER2 positivity rates across clinicopathological subgroups using pooled odds ratios.
2. **Prevalence meta-analysis** — Estimates the pooled prevalence of each HER2 score (0, 1+, 2+, 3+) using the Freeman–Tukey transformation under a random-effects model.
3. **Publication bias assessment** — Evaluates small-study effects via the DOI (Doi) plot and the LFK index.
4. **Meta-regression** — Identifies sources of between-study heterogeneity by fitting a random-effects weighted regression of log odds ratios against study-level covariates (publication year, country, IHC clone, sample size, assessment method, and study design).
