# HER2 expression in prostatic carcinomas – a systematic review and meta-analysis of a potential therapeutic target

## Overview

The analysis pipeline performs three meta-analytic tasks:

1. **Cross-tabulation meta-analysis** — Compares HER2 positivity rates across clinicopathological subgroups using pooled odds ratios with Fisher's exact test.
2. **Prevalence meta-analysis** — Estimates the pooled prevalence of each HER2 immunohistochemistry (IHC) score (0, 1+, 2+, 3+) using the Freeman–Tukey double arcsine transformation under a DerSimonian–Laird random-effects model.
3. **Publication bias assessment** — Evaluates small-study effects via the DOI (Doi) plot and the LFK index.
